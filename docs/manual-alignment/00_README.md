2. `01_GAP_ANALYSIS.md` — background review (manual vs. current, the honest scorecard).
3. `UPGRADE_MAP.md` — the same corrections viewed by nav section.
4. `02_UPGRADE_PLAN.md` / `03_CODE_HANDOFF.md` — deeper detail used only at the "Do" stage (incl. the safety-branch workflow so the live system can always be reverted).
