# Document Library Prompt

Reformat PredictSafeBIO document library materials in a CSEP-style standard.

## CSEP-Style Pattern

1. Section Title
2. Risk / Purpose
3. Required Controls
4. How Controls Are Met and Verified
5. Stop-Work / Hold-Point Triggers
6. Applicable References
7. Document Control / Revision History

## Requirements

- Preserve document logic and gap rules.
- Fix spacing, orphan headings, table overflow, page breaks, and guardrail placement.
- Keep documents readable at normal zoom.
- Keep all AI-drafted document updates labeled draft-only.

## Guardrails

No generated document may claim approval, compliance certification, regulatory acceptance, validation, or CAPA closure.
