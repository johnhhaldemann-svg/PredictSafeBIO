# BioType Foundation Prompt

Implement or review BioType Foundation Packages for PredictSafeBIO.

## Required BioTypes

- R&D Biotech
- Diagnostics / Clinical Lab Support
- Cell & Gene Therapy
- Biologics / Pharma Manufacturing Support
- Medical Device / Diagnostics Manufacturing
- Cleanroom / Controlled Environment
- CRO / Lab Services
- Academic / University Research
- Lab Construction / Commissioning

## Required Behavior

- Allow one primary BioType.
- Allow multiple secondary BioTypes.
- Combine applicable programs, documents, records, risks, and training from all selected branches.
- Preserve source traceability from BioType selection into AI Engine recommendations.

## Acceptance Criteria

- No duplicate merged requirements.
- BioType gaps influence missing-information detection.
- Recommendations remain draft-only and human-review required.
